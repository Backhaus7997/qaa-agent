**9. Troubleshooting (20-30 lines)**
```
## Troubleshooting

### "gh: not authenticated"
Run `gh auth login` and follow the prompts. Select GitHub.com, HTTPS, and authenticate via browser.

### "No git remote found"
The target repository must have a remote origin configured. Run `git remote add origin <url>`.

### "A branch named 'qa/auto-...' already exists"
A pipeline was previously run on the same day. The system auto-appends a suffix (-2, -3), but you can also delete the old branch: `git branch -D qa/auto-...`

### Pipeline stalls at a checkpoint
Type your response in the terminal. For auto-advance mode, use `--auto` flag: `/qa-start --dev-repo <path> --auto`

### Tests fail validation after 3 fix loops
The validator attempted 3 auto-fix cycles. Review VALIDATION_REPORT.md for unresolved issues. Fix manually, then re-run `/qa-validate`.
```
